Hello,
This is a github that analyses the risk factors of the Chronic_Kidney_Disease Data Set. 
It also tries to exctract sub attributes that may be a cause to the CKD 

Please run the jupyter notebook AnalyseDonnées to start

The jupyter notebook communicates with pure python code found in the folder "PythonPart"

The pure python code is used to write functions and modularize the process 

The jupyter notebook is used for the execution of the whole process and for Data visualization


Instructions et commentaires (FR)

Extraction des donneés
L'extraction des données se charge d'importer la base de données et valider son contenu

Traitement des données manquantes
Le traitement des données manquantes est une partie nécessaire pour remplacer les données manquantes par des valeurs de la base de données

Visualisation des données

Analyse des données
L'analyse des données est la partie la plus complexe de ce jupyter notebook, elle se distingue par 3 approches :
- Analyse par différences 
- Analyse par corrélation
- Analyse par régression logistique
